श्रीगणेशाष्टोत्तरशतनामावलिः

ॐ गजाननाय नमः।
ॐ गणाध्यक्षाय नमः।
ॐ विघ्नराजाय नमः।
ॐ विनायकाय नमः।
ॐ द्वैमातुराय नमः।
ॐ द्विमुखाय नमः।
ॐ प्रमुखाय नमः।
ॐ सुमुखाय नमः।
ॐ कृतिने नमः।। ९ ।।

ॐ सुप्रदीपाय नमः।
ॐ सुखनिधये नमः।
ॐ सुराध्यक्षाय नमः।
ॐ सुरारिघ्नाय नमः।
ॐ महागणपतये नमः।
ॐ मान्याय नमः।
ॐ महाकालाय नमः।
ॐ महाबलाय नमः।
ॐ हेरम्बाय नमः।। १८ ।।

ॐ लम्बजठरायै नमः।
ॐ ह्रस्व ग्रीवाय नमः।
ॐ महोदराय नमः।
ॐ मदोत्कटाय नमः।
ॐ महावीराय नमः।
ॐ मन्त्रिणे नमः।
ॐ मङ्गल स्वराय नमः।
ॐ प्रमधाय नमः।
ॐ प्रथमाय नमः।। २७ ।।

ॐ प्राज्ञाय नमः।
ॐ विघ्नकर्त्रे नमः।
ॐ विघ्नहर्त्रे नमः।
ॐ विश्वनेत्रे नमः।
ॐ विराट्पतये नमः।
ॐ श्रीपतये नमः।
ॐ वाक्पतये नमः।
ॐ शृङ्गारिणे नमः।
ॐ अश्रितवत्सलाय नमः।। ३६ ।।

ॐ शिवप्रियाय नमः।
ॐ शीघ्रकारिणे नमः।
ॐ शाश्वताय नमः।
ॐ बल नमः।
ॐ बलोत्थिताय नमः।
ॐ भवात्मजाय नमः।
ॐ पुराण पुरुषाय नमः।
ॐ पूष्णे नमः।
ॐ पुष्करोत्षिप्त वारिणे नमः।। ४५ ।।

ॐ अग्रगण्याय नमः।
ॐ अग्रपूज्याय नमः।
ॐ अग्रगामिने नमः।
ॐ मन्त्रकृते नमः।
ॐ चामीकरप्रभाय नमः।
ॐ सर्वाय नमः।
ॐ सर्वोपास्याय नमः।
ॐ सर्व कर्त्रे नमः।
ॐ सर्वनेत्रे नमः।। ५४ ।।

ॐ सर्वसिद्धिप्रदाय नमः।
ॐ सिद्धये नमः।
ॐ पञ्चहस्ताय नमः।
ॐ पार्वतीनन्दनाय नमः।
ॐ प्रभवे नमः।
ॐ कुमारगुरवे नमः।
ॐ अक्षोभ्याय नमः।
ॐ कुञ्जरासुर भञ्जनाय नमः।
ॐ प्रमोदाय नमः।। ६३ ।।

ॐ मोदकप्रियाय नमः।
ॐ कान्तिमते नमः।
ॐ धृतिमते नमः।
ॐ कामिने नमः।
ॐ कपित्थपनसप्रियाय नमः।
ॐ ब्रह्मचारिणे नमः।
ॐ ब्रह्मरूपिणे नमः।
ॐ ब्रह्मविद्यादि दानभुवे नमः।
ॐ जिष्णवे नमः।। ७२ ।।

ॐ विष्णुप्रियाय नमः।
ॐ भक्त जीविताय नमः।
ॐ जितमन्मधाय नमः।
ॐ ऐश्वर्यकारणाय नमः।
ॐ ज्यायसे नमः।
ॐ यक्षकिन्नेर सेविताय नमः।
ॐ गङ्गा सुताय नमः।
ॐ गणाधीशाय नमः।
ॐ गम्भीर निनदाय नमः।। ८१ ।।

ॐ वटवे नमः।
ॐ अभीष्टवरदाय नमः।
ॐ ज्योतिषे नमः।
ॐ भक्तनिधये नमः।
ॐ भावगम्याय नमः।
ॐ मङ्गलप्रदाय नमः।
ॐ अव्यक्ताय नमः।
ॐ अप्राकृत पराक्रमाय नमः।
ॐ सत्यधर्मिणे नमः।। ९० ।।

ॐ सखये नमः।
ॐ सरसाम्बुनिधये नमः।
ॐ महेशाय नमः।
ॐ दिव्याङ्गाय नमः।
ॐ मणिकिङ्किणी मेखालाय नमः।
ॐ समस्त देवता मूर्तये नमः।
ॐ सहिष्णवे नमः।
ॐ सततोत्थिताय नमः।
ॐ विघातकारिणे नमः।। ९९ ।।

ॐ विश्वग्दृशे नमः।
ॐ विश्वरक्षाकृते नमः।
ॐ कल्याणगुरवे नमः।
ॐ उन्मत्तवेषाय नमः।
ॐ अपराजिते नमः।
ॐ समस्त जगदाधाराय नमः।
ॐ सर्वैश्वर्यप्रदाय नमः।
ॐ आक्रान्त चिद चित्प्रभवे नमः।
ॐ श्री विघ्नेश्वराय नमः।। १०८ ।।

॥ इति श्रीगणेशाष्टोत्तरशतनामावलिः संपूर्णम ॥
